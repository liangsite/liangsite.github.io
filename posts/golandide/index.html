白嫖Goland付费软件

背景

  • 提升编写代码的舒适度以及开发和书写效率。

目的

  • 白嫖使用付费的golang编辑器，通过reset插件可以重置golang编辑器的试用时间，从而达到无限免费使用的目的。

1、添加插件库

  • 打开GolandIDE设置

  • 找到插件Plugins，点击右边小齿轮选择第一个（Manage Plugin Repositories…)

  • 将URLhttps://plugins.zhile.io 添加到仓库列表。

2、安装插件库

  • Marketplace 中搜索 reset 插件 => 安装 IDE EVal Reset插件，重启IDE即可

3、设置自动刷新试用过期时间策略

  • 帮助 => Eval Reset => 勾选上 Auto reset before per restart
